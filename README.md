# PlottingSound
In this project I measured the amplified voltage of a microphone across a transistor, and plotted it on an oscilloscope written in python. The LED chaser kit was powerer by the 5V output of arduino and the analog signal is measured at the A0 pin. In the python code i forgot to change the Y axis to show voltage, to do that you'd need to divide the value by 4095 and muultiply by 5.
